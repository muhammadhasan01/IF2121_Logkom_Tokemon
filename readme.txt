------------- FILE README UNTUK GAME TOKEMON -------------

*** File ini berisi paduan bagaimana untuk memulai game Tokemon ini ***

REQUIREMENTS :

1. Install GNU Prolog
2. File mainProgram.pl
3. File map.pl
4. File player.pl
5. File tokemon.pl
6. File utils.pl
7. File eksternalUtil.pl
8. File fight.pl

HOW TO RUN :
1. Run GNU Prolog
2. Pilih 'File' pada Menu Bar di bagian atas kanan
3. Pilih 'consult'
4. Setelah itu Anda akan diminta untuk memilih file, maka pilihlah file 'mainProgram.pl'
5. Mulailah game dengan command 'Start.'
